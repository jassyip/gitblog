## agency-agents
🔗 **GitHub 仓库**：[https://github.com/msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents)
### 项目概述
**agency-agents** 是一个专注于 AI 专家人格设计的开源项目，致力于构建一套高质量、覆盖广泛的专业 AI 角色库。项目理念在于：通过精心设计的 system prompt，让大语言模型（LLM）能够以特定专业视角和行为逻辑进行思考与输出，从而显著提升模型在垂直领域的表现质量。
### 核心功能
- 收录 **144 个**专业领域的 AI 角色与人格定义，覆盖工程、设计、商业、科学、法律、医疗等多个方向
- 每个角色均配备结构化的 system prompt 模板，包含背景设定、行为准则、输出风格等维度
- 支持直接复用或二次定制，快速适配不同应用场景
- 社区持续贡献与迭代，角色库保持动态更新
### 技术亮点
- **Prompt Engineering 精细化**：每条人格描述均经过反复打磨，强调角色的专业深度与行为一致性，避免模型漂移
- **结构标准化**：统一的角色定义格式，便于批量接入 LLM 应用或 Agent 框架（如 LangChain、AutoGPT 等）
- **轻量无依赖**：纯文本资产，无需额外运行环境，可无缝嵌入任意 AI 工作流
- **多模型兼容**：适用于 GPT-4、Claude、Gemini、开源 LLM 等主流模型
### 使用建议
> 推荐优先将此项目用于需要**专业性输出**的场景，如代码审查、方案设计、学术写作、法律咨询辅助等。
- 使用前先明确目标任务领域，从库中挑选最匹配的角色
- 可将角色 prompt 与 RAG（检索增强生成）结合，进一步提升垂直领域准确率
- 建议针对实际业务需求对原始 prompt 进行微调，而非直接照搬
- 适合作为企业内部 AI 助手或智能客服的人格基础层

## agency-agents-zh
🔗 **GitHub 仓库**：[https://github.com/jnMetaCode/agency-agents-zh](https://github.com/jnMetaCode/agency-agents-zh)
### 项目概述
**agency-agents-zh** 是 agency-agents 的中文本地化增强版，在保留原版专业角色体系的基础上，针对中国市场和中文用户群体进行了深度二次开发。项目不仅完成了角色内容的中文化适配，更在内容策略与平台生态维度上做出了显著扩展，是面向国内 AI 应用场景的重要补充。
### 核心功能
- 在原版 144 个角色基础上，新增**小红书、抖音、微信公众号、B站**等国内主流平台的专属内容角色
- 提供适配中文语境的创作风格模板，涵盖种草文案、短视频脚本、直播话术、私域运营等场景
- 融入中国互联网文化特色，角色表达更贴合国内用户的阅读习惯与审美偏好
- 持续跟进国内平台算法趋势与热点内容策略
### 技术亮点
- **文化语境适配**：不仅是语言翻译，更是对角色底层逻辑的本土化重构，确保内容风格符合国内平台调性
- **平台差异化策略**：针对不同平台的内容规则、用户画像和传播逻辑，提供差异化的角色配置方案
- **运营自动化友好**：角色 prompt 设计充分考虑批量内容生产需求，便于与自动化发布工具集成
- **中英双语兼容**：部分角色支持双语输出，适用于跨境内容团队
### 使用建议
> 特别适合**国内内容创作团队、MCN 机构、电商运营、品牌新媒体**等需要大量本土化 AI 辅助生产的场景。
- 优先用于小红书、抖音等平台的内容批量生产与 A/B 测试
- 与 AI 写作工具（如 Coze、Dify）结合，构建端到端的内容自动化流水线
- 建议定期更新角色库，跟进平台政策和流行内容趋势变化
- 在私域运营场景中，可将角色与用户画像数据结合，实现个性化内容推送

## MiroFish
🔗 **GitHub 仓库**：[https://github.com/666ghj/MiroFish](https://github.com/666ghj/MiroFish)
### 项目概述
**MiroFish** 是一个面向复杂系统研究的多智能体社会仿真预测引擎。项目通过构建多个相互协作与博弈的 AI Agent，模拟真实社会中的群体行为、决策过程与演化路径，从而为社会科学研究、市场分析、政策评估等领域提供数据驱动的仿真支持。
### 核心功能
- **多 Agent 协同仿真**：支持同时运行多个具备独立决策能力的 AI Agent，模拟个体与群体的交互动态
- **社会行为建模**：能够对舆论传播、消费行为、群体决策等典型社会现象进行参数化建模
- **预测分析模块**：基于仿真结果输出趋势预测、敏感性分析和情景推演报告
- **可配置仿真环境**：支持自定义社会变量、Agent 属性和交互规则，灵活适配不同研究场景
### 技术亮点
- **Agent 架构设计**：每个 Agent 具备独立的感知-决策-行动循环，支持异构 Agent 混合仿真，更接近真实社会复杂性
- **LLM 驱动决策**：Agent 的决策逻辑由大语言模型驱动，使其行为更具语义合理性，超越传统基于规则的仿真系统
- **涌现行为捕捉**：系统能够观测并记录宏观涌现现象，揭示个体行为到群体规律的映射关系
- **可解释性输出**：仿真过程可追溯，支持对特定决策节点的行为路径进行回溯分析
### 使用建议
> 适合有一定技术背景的**研究人员、数据科学团队、咨询机构**，以及需要进行前瞻性风险评估的**政策制定机构**。
- 在学术研究中，可用于验证社会学、行为经济学等领域的理论假设
- 企业侧可用于新品上市前的市场反应预测、竞争策略模拟等场景
- 政策评估场景中，建议配合专家知识进行参数校准，以提高仿真结果的可信度
- 初次使用建议从小规模 Agent 场景开始，逐步扩展至复杂系统仿真

## 综合对比与总结

| 维度 | agency-agents | agency-agents-zh | MiroFish |
| --- | --- | --- | --- |
| **项目定位** | 通用 AI 专家人格库 | 中文本土化人格增强版 | 多智能体社会仿真引擎 |
| **核心价值** | 提升 LLM 专业领域输出质量 | 适配国内平台的内容生产 | 复杂社会系统建模与预测 |
| **技术门槛** | ⭐ 低（即插即用） | ⭐ 低（即插即用） | ⭐⭐⭐ 高（需配置与调试） |
| **适用用户** | 开发者、AI 产品团队 | 内容运营、MCN、电商团队 | 研究人员、数据科学家 |
| **主要场景** | 专业咨询、辅助写作、Agent 构建 | 小红书/抖音内容创作、私域运营 | 社会仿真、市场预测、政策评估 |
| **与 LLM 关系** | 输入层（优化 prompt） | 输入层（本土化 prompt） | 决策层（LLM 驱动 Agent） |
| **扩展性** | 高（可自定义角色） | 高（可扩展平台角色） | 中（需一定开发配置） |

### 总结
三个项目在 AI 应用链条上各有侧重，形成互补关系：
- **agency-agents** 与 **agency-agents-zh** 解决的是「如何让 AI 更像专家」的问题，属于 **Prompt 工程基础设施层**，适合快速提升现有 AI 应用的输出专业度，尤其是 zh 版本对国内团队具有直接的落地价值。
- **MiroFish** 则进入了更复杂的「**多 Agent 系统**」领域，核心价值在于通过 AI 模拟人类社会行为，为难以通过实验验证的复杂问题提供仿真路径，是 AI 赋能社会科学与决策支持的前沿探索。
**后续建议**：可考虑将 agency-agents(-zh) 作为 MiroFish 中各 Agent 的人格基础层加以整合，构建具备专业化角色设定的高质量社会仿真系统，实现两类项目的协同增效。
